Welcome to the official repository of **Beacon Academy**. This is a modern, high-performance educational platform and professional portfolio built for speed, accessibility, and seamless content management.

---

## 🚀 Project Overview

Beacon Academy is designed to bridge the gap between technical knowledge and education. It serves as a personal showcase for my research in **Cybersecurity, AI, and Data Science**, while providing a structured learning environment for students.

### 🌟 Key Features

- **⚡ Blazing Fast Performance**: Powered by Astro's Island Architecture, achieving near-perfect Lighthouse scores.
- **🖥️ Integrated Admin Dashboard**: A fully functional CMS (Decap CMS) for publishing content directly from the browser without touching code.
- **📧 Newsletter Integration**: Built-in subscription system using Formspree to keep students updated via email.
- **📚 Rich Content Ecosystem**: Dedicated sections for:
  - **Tech Tips**: Quick, actionable technical insights.
  - **Lessons**: Structured educational modules categorized by difficulty.
  - **eBooks & Templates**: Digital resources for learners and developers.
  - **Projects**: Showcase of engineering and research work.
- **🎨 Modern Aesthetics**: Clean, responsive UI with glassmorphism effects and smooth transitions using Tailwind CSS 4.0.

## 🛠️ Tech Stack

- **Core**: [Astro 5](https://astro.build/) (Static Site Generation)
- **UI Framework**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Content Management**: [Decap CMS](https://decapcms.org/) with [Cloudflare Workers](https://workers.cloudflare.com/) for Auth.
- **Newsletter**: [Formspree](https://formspree.io/)
- **Deployment**: [GitHub Pages](https://pages.github.com/) & [GitHub Actions](https://github.com/features/actions).

## 💻 Local Development

To set up the project locally for development:

1. **Clone & Navigate**:
   ```bash
   git clone https://github.com/rose1996iv/beaconacademy.git
   cd beaconacademy
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Launch Dev Server**:
   ```bash
   npm run dev
   ```
   *Visit `http://localhost:4321` to see the site in action.*

4. **Build for Production**:
   ```bash
   npm run build
   ```
